# Simulasi Jaringan Gedung 3 Lantai
Project ini merupakan simulasi jaringan komputer gedung 3 lantai
menggunakan Cisco Packet Tracer dengan segmentasi jaringan berbasis VLAN
dan routing dinamis antar lantai.

## Topologi Jaringan
- Gedung terdiri dari 3 lantai
- Setiap lantai memiliki router dan switch
- Segmentasi jaringan berdasarkan divisi menggunakan VLAN

## Teknologi yang Digunakan
- Cisco Packet Tracer
- VLAN (10, 20, 30, 40, 50, 60, 70, 80)
- DHCP
- OSPF
- SSH

## Fitur Utama
- IP address otomatis menggunakan DHCP
- Routing dinamis antar lantai menggunakan OSPF
- Segmentasi jaringan tiap divisi
- Keamanan akses perangkat menggunakan SSH

## Cara Menjalankan Project
1. Buka file `.pkt` menggunakan Cisco Packet Tracer
2. Jalankan simulasi
3. Lakukan pengujian konektivitas dengan ping

## Tujuan Project
Project ini dibuat sebagai portofolio
untuk posisi IT Support / IT Infrastructure Intern.
